# Placement Portal

A full-stack MERN Placement Portal that connects students and recruiters through a secure and efficient recruitment platform.

## 🚀 Features

### Authentication
- User Registration
- User Login
- Password Hashing using bcrypt
- JWT Authentication
- Role-Based Authorization (Student, Recruiter, Admin)

### Job Management
- Recruiter can create jobs
- View all jobs
- View job by ID

## 🛠️ Tech Stack

### Frontend
- React
- Vite
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcryptjs

## 📂 Project Structure

```
placement-portal
├── backend
└── frontend
```

## ⚙️ Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
